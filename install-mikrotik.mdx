#### How to install Mikrotik on your EDIS Virtual Server In order to be able to install Mikrotik on EDIS KVM 1\. head to KVM settings in the [services](https://manage.edis.at/whmcs/clientarea.php?action=services) panel 2\. **change Disk driver to IDE** and 3\. **change** **Network Driver to E100**
